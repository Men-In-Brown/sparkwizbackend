What is the purpose of this project?

  1. Share and connect with people on which classes you are taking!

How are we making this project

  1. We are using the implementation of a Spring Java Backend which involves multiple SQLite databases
  2. We incorporated the use of user roles such as Admin, Teacher, and Student
  3. We involve the use of API and API controllers in order for students to create accounts
